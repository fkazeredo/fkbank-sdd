## Findings

See `.claude/runtime/SPEC-0018/qa-report.md` §Findings for full detail and repro commands.

1. **[BLOCKER]** Self-referencing redirects (`/login`, re-redirect to `/oauth2/authorize`) drop
   the port through the nginx edge (`proxy_set_header Host $host;` strips it, and the backend
   has no forwarded-header trust configured). Breaks AC3b, AC5, and — once pushed — AC7.
2. Investigated, resolved as not a defect: `ProductionSecretsGuard`'s apparent single-property
   coverage is correct given `application-prod.yml`'s defaults and `SeededLoginConfig`'s profile
   restriction.

**FAIL_REWORK** — see qa-report.md for full reasoning.
